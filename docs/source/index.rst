QEEG Documentation
==================

QEEG is a Python package for analyzing EEG data, with a focus on quantitative EEG (QEEG) analysis.

Features
--------

* Preprocessing of EEG data (filtering, artifact removal)
* Spectral analysis (power spectral density, band powers)
* Epileptiform activity detection
* Visualization tools (spectra, topographic maps, brain activation)
* Machine learning tools for EEG analysis
* Condition-specific analysis (ADHD, anxiety, autism, depression)
